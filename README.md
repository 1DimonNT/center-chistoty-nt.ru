## 🚀 Особенности реализации

### ✅ Семантическая вёрстка
- Чистая структура заголовков H1-H2
- Обёртка таблиц в `.table-wrapper` для мобильных

### ✅ Адаптивный дизайн
- Медиа-запросы для планшетов и телефонов
- `loading="lazy"` для всех изображений

### ✅ Умная форма заказа
- Заполнение формы → копирование текста в буфер → переход в VK
- Пользователю не нужно ничего печатать вручную в мессенджере

### ✅ Видеоотзывы
- Вставка Rutube-видео через iframe
- Блок `<details>` для дополнительных видео (экономия трафика)

### ✅ Полная микроразметка Schema.org
- `LocalBusiness`, `Service`, `Article`, `FAQPage` — полное покрытие для расширенных сниппетов в выдаче
